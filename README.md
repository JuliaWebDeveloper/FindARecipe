# FindARecipe

**Find a recipe** - приложение, которое дает возможность по ингредиентам найти различные блюда на любой вкус.
***
Приложение поключается к API с различной едой и рецептами.

В данном приложении пользователь в поле ввода указывает интересующие его ингредиенты. 

Далее перед ним появляется список из рецептов, в которых есть нужный пользователю ингредиент.

В приложении представлены фото блюда, все игредиенты, из которых можно приготовить данное блюдо, а также КБЖУ и общий вес блюда.

<h3>Используемые технологии:</h3>
